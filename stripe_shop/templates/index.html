{% load static %} Django + Stripe Checkout

Инструкция

  1. Get Item: Возвращает информацию об объекте.

    HTTP Method: GET

    Endpoint: /item/1

    Описание: Вы можете узнать информацию об объекте с ID 1.

  2. Buy Item: Возвращает sessionId.

    HTTP Method: GET

    Endpoint: /buy/1

    Описание: Stripe Session Id для оплаты выбранного Item.

  3. Config: Возвращает Config Stripe.

    HTTP Method: GET

    Endpoint: /config/

    Описание: Возвращает Config Stripe.

  4. Order: Возвращает информацию об Order Id.

    HTTP Method: GET

    Endpoint: /order/1

    Описание: Возвращает страницу с заказом.
